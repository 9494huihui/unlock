测试情况：针对移动端
在iphone SE上测试通过

大体思路：
一、使用bootstrap搭建页面，分为canvas、p、两个单选框三个部分

二、p和两个单选框实现很简单，重点看一下canvas画布

三、canvas画布

1、初始化，画出9个空心圆

2、触摸屏幕时触发touchstart，判断触摸的位置是否在圆内
（1）如果在圆内，填充圆，并将填充过的圆存入数组clickPoint中
（2）尚未填充的圆存入restPoint数组中

3、手指滑动时连续触发touchmove
（1）清空整个画布，重新画出9个空心圆，根据clickPoint数组来填充圆，画出线条
（2）将此刻触摸的位置和restPoint数组中的圆心位置进行比较，决定是否将圆从restPoint中划出到clickPoint中

4、手指离开时触发touchend，根据单选框选中的状态来决定是设置密码还是验证密码
设置密码的情况：
（1）只输入过一次有效密码（至少需要5个点），提示再次输入
（2）已经输入了两次有效密码，比较两次输入是否一样，相同提示“密码设置成功”，不同提示“两次密码不一致”，并重新录入

验证密码的情况：
比较输入的密码和localStorage中存储的是否相同，相同输出“密码正确”，不同输出“密码错误”，并且一直等待用户输入正确为止。
